(4工夫した点(100文字以上)
ハコスコのVRゲームということで、工夫した点は2点あります。
1つ目は、画面にタッチしないようにすること
2つ目は、プレイヤーに後ろを振り返ったり、横を見たりと前後上下左右を向かせるようにさせること

操作方法はすべてプレイヤーの視点によってゲームをすることができます。
例えば、敵を倒す方法は、敵を見続けることで自動的に敵を攻撃します。
また、ジャンプ道は的を探すこと、分かれ道はデバイスの角度を傾けることで道を進むことができるようにしました。
的や宝を探すために正面だけでなく後ろを振り向かなければなりません。
これによって、一切画面をタッチさせずに、プレイヤーに色々な方向を見てもらえるようにしました。

(5)今後の課題(100文字以上)
今後の課題は、2つあります。
1つ目は、道のバリエーションを増やすこと
2つ目は、Y字道に問題形式で出題すること

進む道が3つ（敵出現道、Y字道、ジャンプ道）しか実装できなかったため、
バリエーションが足りずその他の道を増やしたいです。
また、Y字道が「右に進め」や「左に進め」と表示しているだけなので、問題形式でプレイヤーに頭を使わせたいです。
